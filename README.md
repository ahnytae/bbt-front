# BBT Side Project

[CBT 문제은행 프로젝트] 

[2020.03.07 ~ ] 

학원 대상으로 수능기출 문제들을 풀어볼 수 있는 웹을 만드는 중입니다.

## Branch 관리
develop을 개발 브랜치로 최신화된 브랜치입니다.
develop 브랜치를 기준으로 feature 브랜치를 만들어 작업 후 PR 후 merge 방식으로 진행 되고 있습니다.

## 이슈 관리

현재 진행되고 있는 작업은 [Github Issues](https://github.com/ahnytae/bbt-front/issues)에 등록되어 관리하고 있습니다.

## 프로젝트 관리

프로젝트 진행률을 관리하고 있으며 작업 확인은 [Github Projects](https://github.com/ahnytae/bbt-front/projects)에서 확인할 수 있습니다.
